# 📦 Diretrizes Técnicas e Exportação

Este documento garante que todos os assets criados sejam tecnicamente compatíveis com a engine do jogo.

## 📦 Exportação e Formatos

Todos os assets devem ser exportados seguindo estas regras:

- **Formato:** `.png` com fundo transparente.
- **Escala:** Escalonamento múltiplo de 16 ou 32 px.
- **UI:** Quando aplicável, considerar uso de **9-slice** para componentes de interface redimensionáveis.

## 📚 Recomendações Finais

- **Identidade:** Todos os elementos devem priorizar a clareza visual aliada à identidade cultural brasileira.
- **Responsividade:** A pixel art deve ser adaptável, funcionando tanto na exploração (blocos) quanto no combate.
- **Escalabilidade:** O estilo deve permitir transição para versões futuras (2.5D ou HD art) sem perder a essência visual.

*Este documento pode ser estendido com diretrizes de animação, feedback sonoro e comportamento visual (interações).*
