# Lista de Produção de Assets de Arte – Ato 1

Equipe, este documento é a nossa fonte única da verdade para a produção de todos os assets visuais 2D necessários para o Ato 1 de "Eras do Brasil". Cada item listado aqui é um passo crucial para dar vida à primeira experiência do jogador em nosso mundo. O objetivo é garantir consistência, clareza e um fluxo de trabalho otimizado à medida que construímos a atmosfera única da Mata Costeira.

Reforço que todos os assets devem aderir estritamente à direção de arte definida: **"Pixel Art moderna"**. A paleta de cores deve se concentrar em **tons terrosos, verdes profundos e roxos espirituais**, transmitindo a atmosfera **rústica, espiritual e mística** que é a alma do nosso projeto. A aderência a este guia visual não é apenas uma diretriz técnica, mas a base para a imersão do jogador.

A lista a seguir está organizada por categoria e prioridade, começando pelos elementos mais críticos para a jogabilidade e a narrativa. Vamos focar nossos esforços de forma inteligente para construir este mundo bloco por bloco, sprite por sprite.

## 3. Tilesets e Cenários

Eles precisam ser mais do que simples panos de fundo; devem ser ambientes ricos em detalhes que contam uma história, evocam uma atmosfera de mistério e perigo, e suportam a jogabilidade. Cada tileset deve ser construído de forma modular para se encaixar em nosso sistema de "cenas conectadas", permitindo variedade e rejogabilidade.
